# Gate effectiveness

Expectancy conditional on each gate pass/fail. Identify gates that block winners or allow losers.

- **cooldown**: pass n=0 total_pnl=0, fail n=0 total_pnl=0
- **exhaustion_gate**: pass n=0 total_pnl=0, fail n=0 total_pnl=0
- **funding_veto**: pass n=0 total_pnl=0, fail n=0 total_pnl=0
- **lead_gate**: pass n=0 total_pnl=0, fail n=0 total_pnl=0
- **position_exists**: pass n=0 total_pnl=0, fail n=0 total_pnl=0
- **regime_gate**: pass n=0 total_pnl=0, fail n=0 total_pnl=0
- **score_threshold**: pass n=0 total_pnl=0, fail n=0 total_pnl=0
- **whitelist**: pass n=0 total_pnl=0, fail n=0 total_pnl=0

**Ranked candidates:** Gates with high pass_total_pnl and low fail_total_pnl are protective.
